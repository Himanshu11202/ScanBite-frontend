# ScanBite

ScanBite is a premium dark restaurant SaaS frontend built with Next.js App Router, TypeScript, Tailwind CSS, Framer Motion, and Lucide icons.

## Available commands

- `npm install`
- `npm run dev`
- `npm run build`
- `npm run lint`
- `npm run format`

## Project structure

- `app/` — Next.js App Router pages and layouts
- `components/` — reusable UI components
- `features/` — feature-driven sections and widgets
- `hooks/` — custom React hooks
- `services/` — API clients and feature services
- `lib/` — shared helpers
- `constants/` — static navigation and site values
- `types/` — shared TypeScript types
- `store/` — client state stores
- `styles/` — global and theme styles
